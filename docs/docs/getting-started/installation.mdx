# Quick Start

This page will focus on just how to set up a project with the Züs JS SDK. For API documentation, see [the API Reference](/api).

## Install Züs JS SDK

The Züs JS SDK is available as a package on NPM for use with a module bundler:

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem'

<Tabs>
  <TabItem value="npm" label="NPM" default>
    ```sh 
    npm install @zerochain/sdk
    ```
  </TabItem>
  <TabItem value="yarn" label="Yarn">
    ```sh
    yarn add @zerochain/sdk 
    ```
    </TabItem>
</Tabs>
